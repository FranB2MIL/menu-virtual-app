# 🍽️ Menu Virtual App

## 📌 Description
Full Stack web application for online menu and order management.

This project simulates a digital ordering system similar to platforms like PedidosYa, allowing users to browse products and manage orders.

## 🚀 Technologies
- Backend: .NET (C#) - Web API
- Frontend: Angular
- Database: SQL
- ORM: Entity Framework

## ⚙️ Features
- Product/menu management
- Order management
- REST API for data handling
- Frontend-backend integration

## 🧠 What I learned
- Building REST APIs with .NET
- Designing layered architecture
- Frontend development with Angular
- Integrating full stack applications

## 📂 Project Structure
backend/   -> .NET API  
frontend/  -> Angular app
